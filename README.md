# ag

> ag是一个基于Egret引擎所开发的游戏开发库。


## 安装

1. 下载或clone该库，将bin目录中的ag.文件夹拷贝到你的游戏项目中的libs中。
2. 在`egretProperties.json`文件中的`modules`字段加入如下库标记。

```
{
	"name": "ag",
	"path": "./libs/ag"
}
```

## 功能

- 向量
- 音频管理器
